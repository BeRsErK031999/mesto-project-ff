Mesto Russia
Mesto - это интерактивный веб-сайт, где пользователи могут делиться фотографиями мест, в которых они побывали. Этот проект позволяет пользователям редактировать информацию о себе (имя и род занятий), добавлять новые карточки с изображениями, удалять их и ставить лайки.

Проект разработан в рамках учебной программы. От предыдущих проектных работ он отличается более сложным функционалом, включая работу с сервером для сохранения информации о пользователях и карточках, а также использованием асинхронных запросов для обновления данных без перезагрузки страницы.

Технологии
В проекте использованы следующие технологии:

HTML и CSS для структуры и стилизации страницы.
JavaScript (ES6) для интерактивности: открытие/закрытие попапов, добавление карточек на страницу, обновление информации о пользователе и аватара, взаимодействие с сервером через API.
Webpack для сборки модулей.
Git для контроля версий.
